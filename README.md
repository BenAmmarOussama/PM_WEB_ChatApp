# PM_WEB_ChatApp


## Requirements

  * NodeJs
  * Node package manager (npm)
  
  
## Installation
```bash
  * git clone https://github.com/BenAmmarOussama/PM_WEB_ChatApp.git
  * cd nodeChatApp-ChatRooms
  * npm install
```

## Usage

  * run the server : ``` node server/server.js ``` 
    The app will be running on http://localhost:3000/
